# HW_example - OCR

## Test Env
* Ubuntu 22.04, OpenVINO 2023.1.0

## Installation Guide
* Create python virtual env
```sh
python3 -m venv .venv_hw4
```
* Activate & install required packages
```sh
source .venv_hw4/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```
## How to run
```sh
python3 main.py
```
