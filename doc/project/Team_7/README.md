# BSP: Bench Safety Project

벤치프레스 사고 알림 프로젝트

## Purpose

- 벤치프레스 도중 발생하는 사고를 감지하고 알리는 시스템 개발
- 두 가지 시나리오에 대한 구현 목표 설정
    - 위험 상황: 바벨을 더 이상 들지 못해 바벨에 깔린 상황
    - 긴급 상황: 바벨이 목 부근으로 떨어지는 긴급한 상황
